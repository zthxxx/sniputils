Provide
=======

- **conffor**

  - json file reader
  - csv filer reader

- **logsetting** - file base friendly
- **cli** - custom option
- **database**

  - dataset for sql
  - mongoengine for mongo

- **concurrent**

  - async wrapper
  - thead wrapper

- **daemon** - auto run as daemon
- **import_hook** - crazy import and frame hack
- **snippets** - utils code snippet which not as lib
